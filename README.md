# Boots Boutique API Interface

## Tech Stack

 - Spring Boot
 - HTML, CSS
 - Javascript

Here are what you can do:
- Increase or decrease the quantity of a boot
- Delete the boot
- Search boots with these parameters
  - Size (Float, equal)
  - Quantity (Integer, greater than)
  - Type (BootType, equal), 
  - Material (String, equal)
