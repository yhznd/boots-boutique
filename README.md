# Boots Bootique API Interface

## Tech Stack:

 - Spring Boot
 - HTML, CSS
 - Javascript

Here are what you can do
- Increase or decrease the quantity of a boot
- Delete the boot
- Search boots with parameters (only quantity parameters is working as greater than)
